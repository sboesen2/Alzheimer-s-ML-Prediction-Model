Alzheimer's Risk Prediction

About Genetic Data Input

This tool uses specific genetic markers associated with Alzheimer's risk. Please input values for the following genetic markers:

  • APOE genotype (e.g., 0, 1, or 2 for the number of e4 alleles)
  • TOMM40 '523' poly-T repeat length
  • CLU rs11136000 genotype
  • PICALM rs3851179 genotype
  • CR1 rs6656401 genotype

Enter the values as comma-separated numbers in the order listed above. For example: 1,30,1,2,0

Note: This tool is for educational purposes only and should not be used for medical diagnosis. Always consult with a healthcare professional for medical advice.
